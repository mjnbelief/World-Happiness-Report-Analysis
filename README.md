# 🌍 World Happiness Report Analysis
### 📊 A Data Analysis of Global Wellbeing

## 🧠 Project Overview

This project explores the factors that influence happiness across countries using data from the World Happiness Report.

The analysis focuses on the relationship between happiness and several key indicators, including:

- GDP per capita
- Social support
- Healthy life expectancy
- Freedom to make life choices
- Generosity
- Perceptions of corruption

---
 
## 🎯 Objectives

- Identify the factors most strongly associated with happiness.
- Explore correlations between economic and social indicators.
- Analyze changes in happiness scores over time.
- Discover countries with the largest increases or decreases in happiness.

---

## 📂 Dataset

The dataset contains country-level happiness metrics collected across multiple years.

This project uses data from the World Happiness Report.

Source:
- [World Happiness Report](https://worldhappiness.report/)

---

## 🔍 Analysis

The project includes:

- Data cleaning and preprocessing
- Missing value handling
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Outlier detection
- Data visualization using Seaborn and Matplotlib

---

## 📊 Visualizations

Some of the visualizations created in this project include:

- Correlation heatmaps
- Scatter plots with regression lines
- Happiness distribution plots
- Country comparison bar charts
- Time-series trend charts
- Boxplots for outlier analysis

---

## 🔧 Technologies Used

- Python
- Pandas
